# Blockchain demo

Inspired by:
https://ethereum.stackexchange.com/questions/72284/store-data-on-blockchain-via-backend-app

## Use

### Requirements

    npx truffle init

And put your SmartContract into the contracts/ folder

### Server side Use


Start blockchain (server folder)

    npx ganache-cli

Deploy Smartcontract

    npx truffle migrate


### Client Side Use

    node index.js
